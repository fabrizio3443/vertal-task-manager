# Auditing

In this document, we'll cover everything about the appropiate auditing process for the correct development of this app under it's clear privacy goal.

Privacy is a serious matter, so any privacy affirmation that is stated about this app needs to have solid evidence to back that statement up. For that, we are going to use the Test Driven Development further by making every privacy feature follow a strict process in order to verify how much depth it can cover.
