# User Stories

## User Story 01: Create a task

- As a:
User

- I want to:
Create a new task with task name, description, starting date, finishing date

- So that:
I can keep track of my responsibilities

- Priority:
Must Have

## User Story 02: Set up notifications and alarms

- As a:
User

- I want to:
Set up an alarm and notification for an existing task

- So that:
I can be reminded of my chores outside the app

- Priority:
Should Have

## User Story 03: Set up tasks as routines

- As a:
User

- I want to:
Set a task to repeat on specific days of the week

- So that:
I don't have to manually create my weekly routines

- Priority:
Should Have

## User Story 04: Edit or delete a task

- As a:
User

- I want to:
Edit or delete an existing task

- So that:
I can fix or remove a task after creating it

- Priority:
Must Have

## User story 05: Privacy

- As a:
User

- I want to:
Keep my tasks and information stored strictly on my local device by default

- So that:
My private data is never uploaded to any server without my explicit consent

- Priority:
Must have

## User story 06: Promote a local task to collaborative task

- As a:
User

- I want to:
Select a task an promote it to a shared team project

- So that:
My teammates can edit and track progress on that specific item

- Priority:
Should have

## User story 07: Attach files to a task

- As a:
User

- I want to:
Attach files (e.g. images, pdfs) to a task

- So that:
I can also be reminded of important items involving that task

- Priority:
Won't have (This platform is not intended for file uploading for security reasons)

## User story 08: Local app onboarding

- As a:
User

- I want to:
Open the app and start creating tasks without creating an account

- So that:
I can start using the app as a local-only tool without giving up any personal data

- Priority:
Must have

## User story 09: App server authentication

- As a:
User

- I want to:
Log into a team server using only a username and password (no email required)

- So that:
I can access shared team tasks without linking my identity to a third-party email provider

- Priority:
Must have

## User story 10: Oauth authentication

- As a:
User

- I want to:
Log in using Google, GitHub, Yahoo or any other third-party provider

- So that:
I don't have to remember my password every time

- Priority:
Won't have (Excluded to maintain strict zero-telemetry and third-party independence)

## User story 11: Secure keychain login

- As a:
User

- I want to:
Save my server credentials securely in my device's encrypted keychain

- So that:
I stay logged into my team server without having to re-type my password every time I open the app

- Priority:
Should have

## User story 12: Secure fingerprint/Face ID login

- As a:
User

- I want to:
Use my device's biometric sensors (Fingerprint/Face ID) to authenticate my session

- So that:
I can quickly and securely access my tasks without typing a password

- Priority:
Should have

## User story 13: Team leader (server tasks)

- As a:
Team leader

- I want to:
Create a task and delegate it to a specific team member (or more than one member) so that responsibilities are clearly delegated

- Priority:
Should have
